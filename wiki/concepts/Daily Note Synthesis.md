# Daily Note Synthesis

Daily Note Synthesis is a workflow where human-written daily notes stay in
`raw/daily/`, while the agent periodically reads a date range and reports
themes, pending decisions, candidate concepts, and links to existing ADRs or
debriefs.

## Boundary

Daily notes are treated as raw source material because they are the user's
stream of consciousness. The agent reads them, but does not edit them. Useful
patterns can later become pages in `wiki/` or action in `dev/`.
