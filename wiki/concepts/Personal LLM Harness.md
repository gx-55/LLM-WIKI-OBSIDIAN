# Personal LLM Harness

A Personal LLM Harness is the local environment that gives an agent durable
context, file access, operating rules, and repeatable workflows. In this vault,
the harness is built from ordinary files: [[AGENTS]], `.codex/skills/`,
`.codex/commands/`, `raw/`, `wiki/`, and `dev/`.

## Why It Matters

The harness changes the agent from a stateless chatbot into a maintainer of a
specific knowledge system. Its value comes from constraints: the agent knows
which zones are read-only, which zone it owns, which zone is collaborative, and
which commands require plans.
